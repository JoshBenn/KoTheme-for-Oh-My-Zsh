# Instructions
1. Import repository
2. Place the file `kozsh.zsh-theme` into the directory ~/.oh-my-zsh/themes
3. Open the file `.zshrc` in the directory ~/ with a text editor
4. Replace the current theme: `ZSH_THEME='kozsh'`
5. Save changes
